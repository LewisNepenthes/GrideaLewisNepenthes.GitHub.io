Shortcuts for Android

2020-10-05-Mon

Android
Esc = 后退

Print Screen = 截屏（废话）

Ctrl + Esc = 弹出菜单

Ctrl + P = 设置

Ctrl + W = 设置壁纸

Ctrl + H = Nexus 帮助

Ctrl + M = 打开系统菜单中的应用程序项

Ctrl + Alt + Del = 直接重启。。。没任何提示（慎用！！）

Alt + space = 打开Google now

Alt + Tab（Win + Tab） = 和Windows一样，快速切换任务

Win + A = 计数器

Win + B = 浏览器

Win + C = 通讯录

Win + E = 邮件

Win + G = Gmail

Win + L = Google 日历

Win + P = Google 音乐

Win + T = Hangout，在手机上也许是短信

Win+Q = 长拉菜单栏
